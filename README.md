# Simulateur de Prêt Hypothécaire Intelligent

Ce projet est un outil financier complet permettant de calculer les mensualités, la durée ou le montant d'un prêt de manière dynamique.

## Fonctionnalités

- **Calcul Intelligent** : Remplissez 3 champs, le 4ème est déduit automatiquement.
- **Exportation** : Génération de tableaux d'amortissement en PDF et Excel.
- **Visualisation** : Graphique d'évolution du capital restant dû.

## Installation

### Backend (FastAPI)

`uvicorn main:app --reload`

### Frontend (React)

`npm start`
